定义列表

gameghost-doc
根文档容器，与 graydraft-document 同用。
gameghost-card
通用毛玻璃卡片，适合段落、说明、嵌套列表。

引用块使用 blockquote 或 gameghost-quote，与表格、代码块一致采用毛玻璃卡片基元。

// 代码块：pre.gameghost-codeblock
function demo(x) {
  return x * 2;
}
